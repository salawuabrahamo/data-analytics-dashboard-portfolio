# 📊 Data Analytics Dashboard Project

## 📖 Overview
This project showcases end-to-end data analysis and visualization using Excel, SQL and Power BI. It includes two dashboards:

- 📈 Electronic Product Sales Dashboard
- 🏠 Rental/Real Estate Analysis Dashboard

The goal is to extract insights, identify trends, and support decision-making.

---

## 🧰 Tools & Technologies
- Excel / CSV (Data Source & Cleaning)
- SQL (Data Analysis & Aggregation)
- Power BI (Dashboard Visualization)
- GitHub (Project Documentation)
