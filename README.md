# 📊 Data Analytics Dashboard Project

## 📖 Overview
This project showcases end-to-end data analysis and visualization using Excel, SQL and Power BI. It includes two dashboards:

- 📈 Electronic Product Sales Dashboard
- 🏠 Rental/Real Estate Analysis Dashboard

The goal is to extract insights, identify trends, and support decision-making.

---

## 🧰 Tools & Technologies
- Excel / CSV (Data Source & Cleaning)
- SQL (Data Analysis & Aggregation)
- Power BI (Dashboard Visualization)
- GitHub (Project Documentation)

## 📊 Sales Dashboard Insights
- Total Units Sold: 581K  
- Total Orders: 116K  
- Total Revenue: 489.9M  

### Key Findings:
- Discounts negatively impact revenue beyond a certain point  
- Some products consistently outperform others  
- Seasonal trends affect monthly revenue  

---

## 🏠 Rental Dashboard Insights
- Total Listings: 3002  
- Average Price: ₦125.59M  
- Median Price: ₦126.23M  

### Key Findings:
- Property prices vary significantly by city  
- Certain property types dominate listings  
- High-end properties cluster in specific locations

- Removed duplicates

## 🧠 SQL Analysis

- Aggregate revenue and sales
